# CHANGELOG.md

## [v1.3.0] - 2026-04-16

### Added
- Стеклянные (glassmorphism) виджеты погоды, музыки и календаря
- Сюжетный виджет "Мамуля скоро напишет..." с таймером 60 секунд
- Иконка замка 🔒 для визуальной индикации заблокированных приложений
- Акцентный градиентный баннер для сюжетного события
- Глобальный менеджер таймера `MamulyaTimerManager` (Singleton с сохранением состояния)
- Таймер Мамули теперь запускается при старте приложения, а не при входе в Telegram
- Виджет "Мамуля скоро напишет..." отображает реальное оставшееся время в реальном времени
- Сохранение состояния таймера через SharedPreferences (продолжает работу после закрытия приложения)
- Поддержка подписчиков на обновления таймера для виджетов и активностей

### Removed
- CardView-рамки вокруг иконок приложений
- Фоновый CardView у заголовка "Приложения"
- Цветные фоны у виджетов (заменены на стеклянные)

### Fixed
- ClassCastException при касте виджетов (LinearLayout → CardView)
- Несоответствие количества колонок между адаптером и декоратором (4→5)
- Ошибка setColorFilter с неверным типом (Long вместо Int)
- Исправлено дублирование таймера (отдельный в виджете и в TelegramActivity)
- Исправлено несоответствие отображаемого времени в виджете реальному состоянию таймера
- Исправлен сброс таймера при закрытии TelegramActivity

### Performance
- Оптимизирована отрисовка иконок при скролле
- Уменьшено количество перерисовок при обновлении батареи
- Оптимизирована работа таймера через единый менеджер вместо двух независимых
- Уменьшено количество создаваемых потоков (один таймер вместо двух)

## [v1.2.0] - 2026-04-14

### Added
- Система сценариев для диалога с Мамулей 💖 (50% вероятность выбора роли "мама" или "мошенник")
- Таймер 60 секунд перед отправкой первого сообщения от Мамули
- Полный диалог для роли "мама" (10 сообщений бота, 9 ответов пользователя)
- Полный диалог для роли "мошенник" (10 сообщений бота, 9 ответов пользователя)
- Кнопки выбора ответа вместо поля ввода текста (только для диалогов с ботом)
- Статичный чат с сохранением состояния диалога (после завершения кнопки больше не появляются)
- Отображение процентов заряда батареи рядом с иконкой
- 5 видов иконок батареи в зависимости от уровня заряда (0%, 25%, 50%, 75%, 100%)
- Иконка батареи с молнией для состояния зарядки
- Новая иконка сигнала в виде 5 возрастающих полосок
- Обновление батареи в реальном времени (каждую минуту) во всех активностях
- Сортировка чатов по времени последнего сообщения (новые сверху)
- Система проверки обновлений (автоматическая проверка новой версии на GitHub при запуске)
- Диалог обновления (красивый диалог с информацией о новой версии и changelog'ом)

### Changed
- Переработан диалог с Мамулей: теперь строго по заданному сценарию с выбором ответа
- Верхняя панель TelegramActivity: иконка и название Telegram перенесены влево, кнопки меню и поиска — вправо
- Иконки телефона и сообщений в нижней панели изменены на серый цвет (#8D8D93)
- Иконка батареи в статусной панеле заменена на более красивую векторную
- Иконка сигнала заменена на 5 возрастающих прямоугольников
- Цвета кнопок в диалоге обновления адаптированы под стиль Telegram

### Fixed
- Исправлено отображение времени последнего сообщения в списке чатов
- Исправлено обновление сообщений при нахождении внутри чата (автоматическая проверка каждые 2 секунды)
- Исправлено дублирование сообщений при повторном входе в чат
- Исправлено отображение батареи в ChatActivity (добавлена полная статусная панель)
- Исправлен импорт Intent в TelegramActivity
- Исправлена ошибка Unresolved reference 'BuildConfig' (добавлена генерация BuildConfig)

### Removed
- Убрано поле ввода текста в чатах с ботом (заменено на кнопки выбора ответа)
- Убрана иконка Telegram из нижней панели главного экрана

## [v1.1.0] - 2026-04-13

### Added
- Цветная иконка Telegram
- Окно TelegramActivity с симуляцией мессенджера
- Список чатов с аватарами (буквы в цветных кругах)
- Статусы сообщений: отправлено (1 серая галочка), доставлено (2 серые галочки), прочитано (2 синие галочки)
- Автоматические ответы бота с задержкой 2-10 секунд после прочтения
- Сохранение истории сообщений через SharedPreferences + Gson
- Отображение последнего сообщения в списке чатов
- Индикатор новых сообщений (красный кружок с цифрой) в списке чатов
- Верхняя панель в чате с аватаром, именем, поиском и меню
- Своя статусная панель в TelegramActivity (время, сигнал, батарея)
- Анимация всплывания окна TelegramActivity
- Закругленные углы у сообщений (20dp)
- Фон верхней панели чата (#17212B)
- Splash-экран для Telegram (черный фон с иконкой по центру, отображение 1 секунду)

### Fixed
- Исправлено отображение галочек (слева от сообщения, время справа)
- Исправлены круглые аватары (без квадратных артефактов)
- Исправлена двойная галочка (две галочки рядом)
- Группы и каналы больше не отвечают на сообщения
- Статусы сообщений сохраняются и не сбрасываются при выходе из чата
- Исправлено сжатие иконок при переключении между страницами рабочего стола
- Исправлено отображение миниатюр в папках (иконки теперь видны на синем фоне с закругленными краями)

### Removed
- Убраны цифры у папок
- Убран белый фон у миниатюр в папках
- Убрана анимация появления Splash-экрана Telegram

## [v1.0.0] - 2026-04-11

### Added
- Симулятор рабочего стола
- Две страницы рабочего стола с горизонтальным перелистыванием
- Верхняя статусная панель с отображением текущего времени (обновление каждую минуту)
- Прозрачная статусная панель без фона
- Виджет погоды на первой странице (кликабельный, показывает Toast-заглушку)
- Виджет музыки на первой странице (кликабельный, показывает Toast-заглушку)
- Виджет календаря на первой странице (кликабельный, показывает Toast-заглушку)
- Сетка приложений 4x3 (12 иконок на страницу)
- Папки с приложениями (Соцсети, Google, Развлечения)
- Отображение папок в виде сетки миниатюр 3x3 (всегда 9 мест)
- Миниатюры приложений внутри папок без белого фона
- Нижняя панель (dock) с иконками: Телефон, Сообщения, Браузер, Камера
- Индикатор текущей страницы внизу экрана
- Анимация нажатия на иконки нижней панели
- Полноэкранный режим (скрыта системная панель)
- Фон с градиентом в стиле Honor

### Fixed
- Исправлена проблема с дублированием ресурсов цветов
- Исправлена проблема с приватными системными иконками
- Исправлено отображение сетки миниатюр в папках (поверх фона)
- Исправлена кликабельность иконок (теперь клик работает по всей области)

### Performance
- Использование RecyclerView для плавного скролла приложений
- ViewPager2 для плавного перелистывания страниц
