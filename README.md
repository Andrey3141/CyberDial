## 📖 О проекте

**CyberDial** — это игровой симулятор рабочего стола телефона с полноценным встроенным Telegram-клиентом. Приложение создает полную иллюзию использования реального смартфона: рабочие столы с виджетами, иконки приложений, папки, нижняя панель и даже мессенджер с живыми диалогами.

### Почему это круто?

| Обычные лаунчеры | CyberDial |
|------------------|-----------|
| Требуют установку как системное приложение | Работает как обычное приложение-игра |
| Скучный дизайн | Стилизация под телефон в игровой форме |
| Нет виджетов | Красочные виджеты с анимацией |
| Обычные папки | Папки с миниатюрами 3x3 |
| Нет мессенджера | Полноценный Telegram с диалогами |

## ✨ Возможности

### 🏠 Рабочий стол
| Фича | Описание |
|------|----------|
| 📱 **2 рабочих стола** | Горизонтальное перелистывание страниц |
| 🕐 **Реальное время** | Статусная панель с обновлением каждую минуту |
| 🔋 **Статусная панель** | Часы, сигнал (5 полосок), батарея с процентом |
| 🌤️ **Виджеты** | Погода, музыка, календарь (кликабельные) |
| 📁 **Умные папки** | Сетка миниатюр 3x3 внутри папок (Соцсети, Google, Развлечения) |
| 🖱️ **Анимация нажатий** | Плавная анимация при клике |
| 📍 **Индикатор страниц** | Показывает текущую страницу |
| 🔲 **Полноэкранный режим** | Системная панель скрыта |

### 💬 Telegram мессенджер
| Фича | Описание |
|------|----------|
| 💬 **Список чатов** | 8 чатов (личные и групповые) |
| 👤 **Аватары** | Буквы в цветных кругах |
| ✅ **Статусы сообщений** | Отправлено (1 серая) → Доставлено (2 серые) → Прочитано (2 синие) |
| 🔴 **Непрочитанные** | Красный бейдж с цифрой |
| 🕐 **Время сообщений** | Отображается в списке и внутри чата |
| 📊 **Сортировка** | Чаты сортируются по времени последнего сообщения |
| 💾 **Сохранение истории** | Все сообщения сохраняются в SharedPreferences |

### 🤖 Бот Мамуля 💖
| Фича | Описание |
|------|----------|
| ⏲️ **Таймер 60 секунд** | Первое сообщение приходит через минуту после входа в Telegram |
| 🎲 **Случайная роль** | 50% — заботливая мама, 50% — мошенник |
| 📜 **Сценарий "Мама"** | 10 сообщений от бота, 9 вариантов ответа |
| 🎭 **Сценарий "Мошенник"** | 10 сообщений от бота, 9 вариантов ответа |

| 💾 **Статичный диалог** | После завершения диалог не повторяется |
| 📖 **История переписки** | 5 глав переписки (3 месяца истории) |

## 📥 Установка и запуск

### ⚡ Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/yourusername/CyberDial.git
cd CyberDial

# 2. Откройте проект в Android Studio
# File → Open → выберите папку CyberDial

# 3. Дождитесь синхронизации Gradle

# 4. Запустите на устройстве или эмуляторе
# Run → Run 'app' (Shift + F10)
```

### 📦 Требования
- Android Studio Hedgehog (2023.1.1) или выше
- Android SDK API 36
- Gradle 8.0+
- Устройство с Android 7.0 (API 24) или выше

### 🚀 После установки

На экране телефона появится иконка **CyberDial**. Нажмите для запуска симулятора. Для открытия Telegram нажмите на иконку Telegram в dock bar на главном экране.

## 🤖 Telegram бот

### Как работает диалог с Мамулей 💖

1. **Запуск Telegram** → открывается список чатов
2. **Таймер 60 секунд** → через минуту случайный выбор роли
3. **Роль "Мама"** — заботливый диалог с приглашением на ужин
4. **Роль "Мошенник"** — просьба перевести деньги на подарок
5. **Кнопки выбора** → пользователь выбирает готовый ответ
6. **Бот отвечает** → диалог продолжается по сценарию
7. **Завершение** → после последнего сообщения диалог заканчивается

### История переписки (5 глав)

| Глава | Тема |
|-------|------|
| Глава 1 | Первая ночь в новой квартире |
| Глава 2 | Кризис жанра (бессонница) |
| Глава 3 | Забери вещи (синий свитер) |
| Глава 4 | Болезнь (температура 38) |
| Глава 5 | Сегодня (утренний диалог) |

## 🔧 Решение проблем

<details>
<summary>❌ Не компилируется проект</summary>

1. Проверьте версию Gradle: `./gradlew --version`
2. Синхронизируйте проект: `File → Sync Project with Gradle Files`
3. Очистите кэш: `Build → Clean Project`, затем `Build → Rebuild Project`
</details>

<details>
<summary>❌ Не приходит сообщение от Мамули</summary>

1. Подождите 60 секунд после открытия Telegram
2. Проверьте, что вы находитесь в списке чатов (не внутри диалога)
3. Перезапустите приложение и попробуйте снова
</details>

<details>
<summary>❌ Не отображается батарея</summary>

1. Убедитесь, что у устройства есть аккумулятор
2. На эмуляторе может не работать — используйте реальное устройство
3. Проверьте разрешения в AndroidManifest.xml
</details>

<details>
<summary>❌ Кнопки выбора ответа не появляются</summary>

1. Убедитесь, что вы открыли личный чат (Анна, Михаил, Мамуля, Папа, Друг)
2. Для Мамули дождитесь первого сообщения (60 секунд)
3. Проверьте, не завершён ли уже диалог
</details>
